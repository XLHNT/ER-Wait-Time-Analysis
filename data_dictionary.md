# Data Dictionary

| Column Name | Description |
| :--- | :--- |
| **Visit ID** | Unique identifier for each ER visit. |
| **Patient ID** | Unique identifier for the patient. |
| **Urgency Level** | Categorization of patient severity (Critical, High, Medium, Low). |
| **Time to Registration** | Minutes from arrival to registration. |
| **Time to Triage** | Minutes from registration to clinical triage. |
| **Time to Medical Prof.** | Minutes from triage to seeing a doctor/PA. |
| **Total Wait Time** | Total minutes spent waiting across all stages. |
| **Nurse-to-Patient Ratio** | Number of nurses available per patient during the visit. |
| **Specialist Availability** | Count of specialists on-call or available. |
| **Patient Satisfaction** | Score from 1-10 provided by the patient post-discharge. |
| **Patient Outcome** | Final status (Admitted, Discharged, Transferred). |
